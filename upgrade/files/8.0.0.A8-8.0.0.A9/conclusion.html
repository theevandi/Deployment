• Add the following line to 'inc/header.inc.php' file: 
            define('BX_SYSTEM_FFMPEG', BX_DIRECTORY_PATH_ROOT . 'plugins/ffmpeg/ffmpeg.exe'); ///< path to ffmpeg binary
    Before the following line: 
            define('BX_SYSTEM_CONVERT', ...
  • Change permissions to the following file: 
            plugins/ffmpeg/ffmpeg.exe
    Usually it can be done by running the following command via SSH or FTP in the root folder: 
            chmod 755 plugins/ffmpeg/ffmpeg.exe
    Please note: by default ffmpeg for linux is provided, if you are using other OS you need different one, you can get it for Mac OSX and Windows.
